---
title: 'Buscar Pagamento por ID'
description: 'Retorna os detalhes de um pagamento específico'
openapi: 'GET /api/v1/payments/{paymentId}'
---

Retorna as informações completas de um pagamento específico pelo seu ID.

## Endpoint

<CodeGroup>
```bash cURL
curl https://api.upag.io/api/v1/payments/pay_xyz789abc \
  -H "Authorization: Bearer {token}"
```

```javascript SDK
import { Upag } from 'upag';

const upag = new Upag('sk_test_your_api_key');

const payment = await upag.payments.retrieve('pay_xyz789abc');
```
</CodeGroup>

## Parâmetros

<ParamField path="paymentId" type="string" required>
  ID único do pagamento, começando com `pay_`.
</ParamField>
